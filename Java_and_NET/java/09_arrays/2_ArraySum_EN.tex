\subsubsection{Summing elements of array}

Another example:

\begin{lstlisting}[style=customjava]
public class ArraySum
{
	public static int f (int[] a)
	{
		int sum=0;
		for (int i=0; i<a.length; i++)
			sum=sum+a[i];
		return sum;
	}
}
\end{lstlisting}
